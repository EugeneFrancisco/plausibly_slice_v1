Ribbon concordances and slice obstructions: code and data
=========================================================

This is the repository of code and data that accompanies the paper:

* Nathan M. Dunfield and Sherry Gong, "Ribbon concordances and slice
  obstructions: experiments and examples", https://arXiv.org/abs/FILLIN

See the "code" and "data" folders as appropriate.  The quick-access
file of PD codes of the knots mentioned by name in the paper is at the
top level with this README file as:

  knots_mentioned_PD_codes.csv

The roughly 12,000 knots where we were unable to completely resolve
their slice status are given in:

  data/plausibly_unknown.csv

including their PD codes.
